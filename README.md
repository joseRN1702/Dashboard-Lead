# Site Dashboard

Pequeno site estático com páginas HTML, CSS e JS.

Como subir para o GitHub:

1. Inicialize o Git: `git init`
2. Faça o commit inicial: `git add . && git commit -m "Initial commit"`
3. Crie um repositório no GitHub e adicione como remote: `git remote add origin <URL>`
4. Faça push: `git branch -M main && git push -u origin main`

Veja também: `.gitignore` já incluído para ignorar dependências e arquivos temporários.
